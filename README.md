flagger
=======

A hackday application to render Mendeley catalg results as their ascii sempahore flag representations

e.g. "Cloud Computing" would render as:

\     / \   \ /  | 
 o   o  -o   o   o 
 |  /            | 

\   \        |  \ / \|  \          
 o  -o   o- -o   o   o   o   o   o 
 |      /               /   / \  |\
